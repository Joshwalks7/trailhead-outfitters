🧱 Part 2 — Technical Planning

Planned Pages / Views

  • Home Page: Features a hero banner and best sellers products list with a normal header. The products tab opens to show a couple different categories like "Men", "Women", "Gear", etc
  • Product Search Page: Loads products based on the customer's search query in the URL parameters. If a product category is selected it uses that category as the query parameters. Clicking on a product opens a modal view with a larger image, the description, price, quantity, and a button to add it to the cart.
  • Cart / Order Form Page: Items in the cart are loaded using localStorage and are displayed to one side with the cart financial details are on the other. Checking out opens the modal that lets the user enter data into a form

Planned JavaScript Modules

  • main.js: This is where the initialization occurs when the page loads. It will trigger the initial data fetching from the JSON file
  • data.js: This will simply handle fetching data from the JSON file
  • render.js: This will be responsible for rendering the page load with product cards from the data.js file as well as fetching data stored in localStorage on the cart page. Modals will appear and go away, dropdowns will perform as expected
  • storage.js: Manages getting, setting, and deleting items inside localStorage for the shopping cart.

Planned Data Structure

Data schema structure for application components: Data will be fetched from a JSON file that is structured to have a name for the product, the categories it belongs too so that it can be easily searched, the price, description that shows on the modal view, and image url path

{
                "id": "tent-01",
                "name": "Alpenglow 2-Person Tent",
                "categories": ["camp-kitchen", "backpacking"],
                "price": 189.99,
                "description": "A lightweight, three-season backpacking tent with dual vestibules.",
                "imageUrl": "images/tent-01.jpg"
            }

Planned localStorage Feature

Shopping Cart Persistence: I am going to use localStorage to retain a customer's cart data even if they leave the page. Obviously it will eventually go away with time or if the user clears their cache, but it should be fairly convenient. This will allow the user to view their cart and continue shopping.
